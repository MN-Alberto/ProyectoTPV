Registra eventos de auditoría relacionados con la gestión de usuarios.

registrarLogUsuario(PDO $pdo, string $tipo, int|null $usuario_id, string $usuario_nombre, string $descripcion[, mixed|null $detalles = null ]) : void

Almacena quién realizó la acción, sobre qué usuario y una descripción legible.

Parameters
$pdo : PDO

Instancia de conexión activa a la base de datos.

$tipo : string

Categoría técnica del evento (ej: 'alta_usuario', 'cambio_password').

$usuario_id : int|null

ID del usuario sobre el que se actúa (puede ser nulo en logins fallidos).

$usuario_nombre : string

Nombre o login del usuario afectado.

$descripcion : string

Explicación en lenguaje natural de lo sucedido.

$detalles : mixed|null = null

Metadatos adicionales (arrays o JSON) para depuración o historial.

registrarLog()

cLogin.php : 24

Registra un evento de auditoría en la tabla de logs del sistema.

registrarLog(PDO $pdo, string $tipo, string $descripcion[, mixed|null $detalles = null ]) : void

Útil para rastrear inicios de sesión, cierres de sesión e intentos fallidos.

Parameters
$pdo : PDO

Instancia de conexión a la base de datos para realizar la inserción.

$tipo : string

Categoría del log (ej: 'login', 'logout', 'error').

$descripcion : string

Mensaje legible describiendo detalladamente el suceso.

$detalles : mixed|null = null

Metadatos adicionales (navegador, IP, etc.) en formato JSON.

Return values
void —

No retorna valor; los errores se silencian para no interrumpir el flujo del usuario.
